# Project Skebob

[![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)](https://developer.android.com)
[![Kotlin](https://img.shields.io/badge/Kotlin-0095D5?style=for-the-badge&logo=kotlin&logoColor=white)](https://kotlinlang.org/)
[![Material Design](https://img.shields.io/badge/Material%20Design-757575?style=for-the-badge&logo=material-design&logoColor=white)](https://material.io/)

**Project Skebob** — современное Android-приложение для получения, хранения и управления интересными фактами.  
Использует **Retrofit** для работы с API, **SQLite** (локальная БД) для офлайн-доступа, реализует систему избранного, фильтрацию записей за 24 часа и живую статистику.  
Интерфейс построен на **Material Design** и адаптирован под портретную и ландшафтную ориентации.

## 📱 Возможности

- 🔄 Загрузка случайных фактов из сети с сохранением в локальную базу
- ⭐ Добавление фактов в «Избранное» (защита от удаления)
- 🗑️ Удаление обычных фактов свайпом вправо
- 📊 Живая статистика: общее количество фактов, максимальная длина текста
- ⏱️ Фильтрация записей, добавленных за последние 24 часа
- 📱 Адаптивный дизайн для портретной и ландшафтной ориентации (двухпанельная верстка)
- 🗃️ Нижняя навигация: все факты / избранное
- 💾 Полная офлайн-работоспособность

## 🛠️ Технологии

- **Язык:** Kotlin
- **Архитектура:** MVVM
- **Сеть:** Retrofit, Coroutines
- **БД:** SQLite (с миграциями)
- **UI:** Material Design Components, RecyclerView, Swipe gestures
- **Навигация:** Bottom Navigation View

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/An3step/Project_Skebob.git
2. **Откройте проект в Android Studio**<br>

3. **Синхронизируйте Gradle**<br>
Дождитесь автоматической загрузки зависимостей (Retrofit, Coroutines, Material Components и др.).

## 📖 Использование

| Действие | Результат |
|----------|-----------|
| **Кнопка «Загрузить факт»** | Получение случайного факта из сети с отображением в списке и сохранением локально. |
| **Свайп вправо** | Удаление обычного факта (без отметки «Избранное»). |
| **Клик по элементу** | Добавление/удаление факта в «Избранное» (появляется значок ⭐). |
| **Нижняя навигация** | Переключение между всеми фактами и только избранными. |
| **Кнопка фильтра** | Показывает только факты, добавленные за последние 24 часа. |
| **Статистика (верх экрана)** | Автообновление: общее количество фактов и максимальная длина текста в текущем списке. |
| **Поворот устройства** | Автоматическая смена раскладки на двухпанельный вид в ландшафтном режиме. |

> [!TIP]
> Избранные факты **нельзя удалить свайпом**. Чтобы удалить такой факт — сначала снимите отметку «Избранное» простым нажатием.
